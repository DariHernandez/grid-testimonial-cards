# Container

Wrapper for all page content 

# container

Regular container with max with and horizontal align

```html
<!-- index.html -->

...
<body>
    <!-- <div id="root"></div> -->

    <div class="container">
        <h1>Sample text</h1>
    </div>
		...
</body>
</html>
```
